# 3 分钟 Demo 讲稿

## 0:00–0:25｜问题与边界

“这个项目是一个 AI 简历初筛与岗位匹配助手。它不自动决定录用或淘汰，而是把简历和岗位 JD 转为可追溯的结构化结果，帮助招聘人员快速复核。”

## 0:25–1:05｜工作流

“流程由四个节点组成：用户输入接收简历和 JD；LLM 依据明确权重输出候选人分析；Code 节点解析 JSON 并在异常时兜底；输出节点仅返回结构化结果和解析状态。”

## 1:05–1:55｜高匹配测试

“第一份样例具有 Prompt、RAG、API 和自动化项目经验，但缺少 Dify/Coze/n8n 的明确实操。系统给出 72 分和 manual_review，并把缺失经验列为待确认项，而不是直接否定候选人。”

## 1:55–2:30｜信息缺失测试

“第二份样例信息很少。系统给出 2 分和 supplement，提示补充工作经历、项目与相关技能。这验证了信息不足时的风险控制。”

## 2:30–3:00｜总结

“目前已验证结构化输出和 JSON 容错。下一步会补齐剩余样例、加入 Excel 汇总和飞书或 Dify 的人工复核闭环。AI 负责提效，最终决策权始终在人。”
